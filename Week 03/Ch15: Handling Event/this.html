Object Methods: this

Following are the best way to use this:

• We can use this to specifically denote that the instance variable is used instead of static or local variable.
• We can use this to refer the constructors.
• We can use this as a parameter
• We can use this to return the current instance
• We can use this in arrow function. If we use this in arrow function, we take this from outside.

Exercise

What is the result of this code?

let user = {
name: "John",
go: function() { alert(this.name) }
}
(user.go)()

The solution is:

The (user.go) do nothing because we need to close the element before.
And we will have a error. If we do in this way:

user = {
name: "John",
go: function() { alert(this.name) }
};
(user.go)()
We will have the answer “John”.
